# CourseRegistrationSystem
A Java desktop application for managing course registration with separate student, faculty, and database components.
Note : 
a) You can add mysql-connector-j-9.6.0.jar file from CourseRegistrationSystem\lib\mysql-connector-j-9.6.0 into Referenced Libraries to run the program smoothly.
b) You need to use your own credentials of MySQL in DatabaseManager.java User/Pass to access database connection in MySQL.
In Case : databses and tables are unable to be created by the program in MySQL, you can just copy all the sql commands from database.sql file available here and once run in MySQL mannually and then the program will run as it is intended to.
